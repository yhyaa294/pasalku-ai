### **Problem: Not enough disk space**

**Solution:**
```
1. Check available space on C: drive (should have at least 2GB free)
2. If needed, clean up Downloads folder or temp files
3. Or copy to external drive instead of Desktop
```

### **Problem: Can't find pasalku-ai-3 folder**

**Solution:**
```
1. Open PowerShell or Command Prompt
2. Type: dir C:\Users\YAHYA\pasalku*
3. This will show you the exact location
4. Navigate to that location in File Explorer
```
